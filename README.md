> 각 문제에 찜하거나 풀었다는 표시를 할 수 있으며 각각 찜한 리스트와 풀었던 리스트에서 확인 가능합니다.

> 각 문제에 할당된 타이머는 총 <b>3단계</b> (스스로 고민하며 푸는 시간, 힌트를 참고하며 푸는 시간, 해설하는 시간)로 존재합니다. 각 단계에서 시간 내에 완료되었다면 건너뛰기 할 수 있습니다. 데모 특성 상 각 단계를 10초 이내로 설정하였습니다.
